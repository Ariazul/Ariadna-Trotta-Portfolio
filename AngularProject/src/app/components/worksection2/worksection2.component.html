Calm Island

Clam Island es un videojuego de exploración diseñado y desarrollado por un equipo de dos artistas y técnicos, quienes actualmente seguimos trabajando para alcanzar una versión beta. Las herramientas utilizadas para la creación de este proyecto fueron principalmente Unity como motor de videojuegos, Blender como herramienta adicional para modelado y texturas, y Wwise como middleware de audio. En el walkthrough inicial pueden apreciarse la implementación de ambientes generativos y adaptativos, curvas de atenuación y filtros responsivos a la distancia, sistemas de pasos adaptativos, scripts de audio en C#, música generativa y espacios reverberantes.
